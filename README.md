# Advanced Linguistic Functions Exercices

## 1. Delegates_InteractiveMusicPlayer
This is a console application built with delegates and events using .NET Core 9.0
The application prompts the user for a song title and an action, which could be:
Play, Pause, Stop, Skip, Exit

Esta es una aplicación de consola creada con delegados y eventos con .NET Core 9.0
La aplicación pide un título de canción y una acción al usuario que podría ser:
Play, Pause, Stop, Skip, Exit

## 2. Events&Delegates_MathematicalOperations

This is a console application built with delegates and events using .NET Core 9. The application performs addition, subtraction, multiplication, and division operations using methods within the MathOperations class, which contains an event handler for each method. The application then displays the results using a delegate to support another display method within the ResultDisplay class.

Esta es una aplicación de consola creada con delegados y eventos con .NET Core 9.
La aplicación realiza operaciones de suma, resta, multiplicación y división con métodos dentro de la clase MathOperations, la cuál contiene un eventos para cada método, y muestra el resultado con la ayuda de un delegado adecuado para admitir otro método de visualización dentro de la clase ResultDisplay.

## 3. LINQ_DataManipulation

Starting with a Person class having three properties (name, age, and salary), which are entered via the console, five LINQ queries are implemented to:

- Query 1: Add a new person
- Query 2: Display a complete list of people
- Query 3: Filter by age, showing only those older than 30
- Query 4: Sort by the first letter of the name
- Query 5: Calculate the average salary
- Query 5: Calculate the total age


A partir de una clase Person con 3 propiedades (nombre, edad y salario) que se ingresan en consola, se implementan 5 consultas LINQ para :

- Consulta 1: Agrega una nueva persona
- Consulta 2: Muestra la lista completa de personas
- Consulta 3: Filtrar por edad, mostrando solo las personas mayores de 30 años
- Consulta 4: Ordenar por la primera letra del nombre
- Consulta 5: Calcular el salario promedio
- Consulta 6: Calcular la edad total
